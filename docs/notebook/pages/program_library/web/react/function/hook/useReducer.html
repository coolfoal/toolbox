useReducer

一个用于状态管理的 hook api，useState 是使用其构建的。

语法 

let [state, dispatch] = useReducer(reducer, initialArg, init?)
  • reducer： 用于更新 state 的纯函数。参数为 state 和 action，返回值是更新后的 state。state 与 action 可以是任意合法值。
  • initialArg： 用于初始化 state 的任意值。
  • init： 可选，用于计算初始值的函数。如果存在，使用 init(initialArg) 的执行结果作为初始值，否则使用 initialArg。

一个简单的例子

比如我们有个两个按钮，其中一个用于数据加一，一个用于数据减一： 

import { useReducer } from 'react'
function Demo(){
    let [count, dispatch] = useReducer(function(state, action){
        if(action == "increment") return state + 1
        else if(action == "decrement") return state - 1
        else return state
    }, 0)

    return (<div>
        记数：{count}
        <button onClick={()=>dispatch("increment")}>增加</button>
        <button onClick={()=>dispatch("decrement")}>减少</button>
    </div>)
}
