如何定义类组件？

基本结构 

class Demo extends React.Component {
    constructor(props) {
        super(props)
    }
    render() {
        return (

            页面内容部分
        
)
    }
}

插槽

比如我们打算写个组件Demo，用起来大概是这样子： 

<Demo>用户自定义的一些内容</Demo>

如果我们希望组件中间的内容在组件的指定位置显示，就可以： 

class Demo extends React.Component {
    private slots: Array
    constructor(props) {
        super(props)
        this.slots = React.Children.toArray(props.children)
    }
    render() {
        return (

            页面内容部分
            {slots.map(slot => {
                return slot
            })}
        
)
    }
}

渲染结果就是： 


    页面内容部分
    用户自定义的一些内容

当然，你也可以通过判断slot的一些属性来实现类似具名插槽的效果。
