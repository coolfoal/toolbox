基础&核心

ref

用于定义基本数据类型的响应式数据： 

<template>
    {{num}} 
</template>
<script setup lang="ts">
    import { ref } from 'vue';
    let num = ref(100);

    // setup中使用.value来访问
    num.value = 101;
</script>

reactive

定义引用数据类型的深度响应式变量。

定义引用数据类型的深度响应式变量。

reactive的一种巧妙使用

首先定义 form 变量，其中包含多个属性：name, age, sex。

假如需要 item 给 form 赋值，我们可能会觉得 Object.assign 是种比较简便的方法：

let form = reactive({
    name: '',
    age: '',
    sex: ''
});
let item = { name: 'Alice', age: '19', sex: 'F' };
Object.assign(form, item);

上面的写法，其实是不优雅的写法。

更优雅的做法，是在定义 reactive 时使用 item 包裹 name、age、sex 属性：

let form = reactive({
    item: {
        name: '',
        age: '',
        sex: '',
    }
});
let item = { name: 'Alice', age: '19', sex: 'F' };
form.item = item;
