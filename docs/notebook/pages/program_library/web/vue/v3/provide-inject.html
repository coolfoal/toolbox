依赖注入

支持组件为后代组件提供数据或方法等，比如我们有如下两个自定义组件的用法： 

<p-demo>
    <c-demo></c-demo>
</p-demo>

我们现在希望通过依赖注入实现 p-demo给c-demo 传递数据，怎么办？

Provide (提供)

很明显，p-demo提供数据，大致写法如下： 

import { provide } from 'vue';
provide("key", "来自PDemo的数据");

Inject (注入)

现在，就可以在c-demo消费这个数据了： 

import { inject } from 'vue';
let value = inject("key");
