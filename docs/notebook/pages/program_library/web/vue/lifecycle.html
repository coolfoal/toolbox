Vue2 Options-based API Vue Composition API 执行时机
beforeCreate setup() 在实例初始化之后,进行数据侦听和事件/侦听器的配置之前同步调用。
created 在实例创建完成后被立即同步调用。
beforeMount onBeforeMount 在挂载开始之前被调用，相关的 render 函数首次被调用。
mounted onMounted 实例被挂载后调用。
beforeUpdate onBeforeUpdate 在数据发生改变后，DOM 被更新之前被调用。
updated onUpdated 在数据更改导致的虚拟 DOM 重新渲染和更新完毕之后被调用。
beforeDestroy onBeforeUnmount 实例销毁之前调用。
destroyed onUnmounted 实例销毁后调用。
errorCaptured onErrorCaptured 在捕获一个来自后代组件的错误时被调用。
activated onActivated 被 keep-alive 缓存的组件激活时调用。
deactivated onDeactivated 被 keep-alive 缓存的组件失活时调用。
