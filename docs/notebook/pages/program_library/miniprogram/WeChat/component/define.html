自定义组件

基本结构 

Component({

    // 属性列表
    properties:{}

    // 数据
    data:{},

    // 自定义方法(位置)
    methods:{},

    // 生命周期方法(位置)
    lifetimes:{
        ready(){}
    },
});

属性定义和使用

属性也就是用户使用这个组件的时候可以设置的属性： 

Component({
    properties: {
        width: {
            type: Number,
            value: 300
        }
    }
});

使用的时候也非常简单： 

    console.log(this.data.width);
