全局配置 app.json 
{
    "pages": [
        "pages/index/index",
        "pages/logs/logs"
    ],
    "window": {
        "navigationBarTitleText": "微信小程序"
    }
}

entryPagePath

指定小程序的默认启动路径（首页）。

window

用于设置小程序的状态栏、导航条、标题、窗口背景色。

navigationBarTitleText

导航栏标题文字内容。

enablePullDownRefresh

是否开启全局的下拉刷新。
