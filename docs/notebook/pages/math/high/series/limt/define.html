基本概念与性质

定义 设{an}为数列，a为定数。若对任给的正数ε，总存在正整数N，使得当n＞N时有 |an - a| ＜ ε 则称为数列{an}收敛于a，定数a称为数列{an}的极限，并记作 ["limt","n→∞",["rightBottom","a","n"]] 读作“当n趋于无穷大时，{an}的极限等于a”。

重要性质

定理（唯一性） 若数列{an}收敛，则它只有一个极限。

定理（有界性） 若数列{an}收敛，则{an}为有界数列，即存在正数M，使得对一切正整数n有|an| ≤ M。

定理（保号性） 若 ["limt","n→∞",["rightBottom","a","n"]] = a＞0(或＜0)，则对任何a'∈(0,a)(或a'∈(a,0))，存在正数N，使得当n＞N时有an＞a'(或an＜a')。

定理（保不等式性） 设{an}与{bn}均为收敛数列。若存在正数N0，使得当n＞N0时有an≤bn，则 ["limt","n→∞",["rightBottom","a","n"]] ≤ ["limt","n→∞",["rightBottom","b","n"]]。

定理（迫敛性） 设收敛数列{an},{bn}都以a为极限，数列{cn}满足：存在正数N0，当n＞N0时有 an≤cn≤bn 则数列{cn}收敛，且 ["limt","n→∞",["rightBottom","c","n"]] = a。
