什么是矩阵？

在平面坐标系中，有一个点 ["join",["rightTop","p","'"],"(",["rightTop","x","'"],",",["rightTop","y","'"],")"] 经过逆时针旋转θ度，那么坐标变换公式就是：

["equationSet", ["join","x=",["rightTop","x","'"],"cosθ - ",["rightTop","y","'"],"sinθ"], ["join","y=",["rightTop","x","'"],"sinθ + ",["rightTop","y","'"],"cosθ"] ]

显然，新旧坐标之间的关系完全可以通过公式中系数所排成的2ㄨ2的块：

["matrix",[["cos","-sinθ"],["sinθ","cosθ"]]]

来表示，而这种块，我们就称为矩阵。

当然，矩阵可以有任意行或列，这里举的例子是一个2行2列的矩阵。

如何表示矩阵？

我们用大写的拉丁字母A,B,...，或者(aij),(bij),...来表示矩阵。

为了指明所讨论的矩阵的级数，可以把sㄨn矩阵写成Asn,Bsn,...，或者(aij)sn,(bij)sn,...

如何判断矩阵相等？

设 A = (aij)mn，B = (bij)lk，如果满足下面两个条件：

  1. m = l 且 n = k；
  2. aij = bij ，对i=1,2,...,m;j=1,2,...,n都成立。

我们就说A=B，即只有完全一样的矩阵才叫做相等矩阵。
