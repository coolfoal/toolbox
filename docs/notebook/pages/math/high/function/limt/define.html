定义（趋向正无穷大） 设ƒ为定义在[a,+∞)上的函数，A为定数。若对任给的ε＞0，存在正数M（≥ a），使得当x＞M时有 |ƒ(x) - A|＜ε 则称函数ƒ当x趋于+∞时以A为极限，记作 ["limt","x→+∞","ƒ(x)"] = A
