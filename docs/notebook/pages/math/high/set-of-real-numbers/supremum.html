确界

什么是确界？

比如，如果一个数集有上界，那么它有无数个上界，而其中最小的那个，就叫该数集的上确界，最大的下界叫下确界。

下面，我们来给出准确的数学描述：

定义 设S是R中的一个数，若数η满足：

  1. 对一切x∈S，有x≤η，即η是S的一个上界；
  2. 对任何α＜η，存在x0∈S，使得x0＞α，即η是S的最小上界。

则称数η为数集S的 上确界 ，记作 η = sup S。

定义 设S是R中的一个数，若数η满足：

  1. 对一切x∈S，有x≥η，即η是S的一个下界；
  2. 对任何α＞η，存在x0∈S，使得x0＜α，即η是S的最大下界。

则称数η为数集S的 下确界 ，记作 η = inf S。

相关定理

定理（确界原理） 设S为非空数集。若S有上界，则S必有上确界；若S有下界，则S必有下确界。
