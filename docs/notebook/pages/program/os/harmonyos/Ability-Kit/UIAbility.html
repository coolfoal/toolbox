@ohos.app.ability.UIAbility

UIAbility是包含UI界面的应用组件，继承自 @ohos.app.ability.Ability ，提供组件创建、销毁、前后台切换等生命周期回调，同时也具备组件协同的能力。

简单的理解就是：一个APP中一个独立的窗口，每个窗口可以有多个页面。

定义

比如我们定义一个入口 EntryAbility 窗口，大致如下： 

import UIAbility from '@ohos.app.ability.UIAbility';

export default class EntryAbility extends UIAbility {
    onWindowStageCreate(windowStage: window.WindowStage) {
        windowStage.loadContent('pages/Index', (err, data) => {
            // todo
        });
    }
}

生命周期

onCreate

实例处于完全关闭状态下被创建完成后进入该生命周期回调，执行初始化业务逻辑操作： 

    onCreate(want: Want, launchParam: AbilityConstant.LaunchParam): void

onWindowStageCreate

当WindowStage创建后调用： 

    onWindowStageCreate(windowStage: window.WindowStage): void

onWindowStageWillDestroy

当WindowStage即将销毁时调用： 

    onWindowStageWillDestroy(windowStage: window.WindowStage): void

onWindowStageDestroy

当WindowStage销毁后调用： 

    onWindowStageDestroy(): void

onWindowStageRestore

当迁移多实例ability时，恢复WindowStage后调用： 

    onWindowStageRestore(windowStage: window.WindowStage): void

onDestroy

在销毁时回调： 

    onDestroy(): void | Promise<void>

onForeground

当应用从后台转到前台时触发： 

    onForeground(): void

onBackground

当应用从前台转到后台时触发： 

    onBackground(): void

onBackPressed

当UIAbility侧滑返回时触发： 

    onBackPressed(): boolean
