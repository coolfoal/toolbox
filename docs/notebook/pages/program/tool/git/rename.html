文件或文件夹名称大小写修改

比如我们有个文件 ./src/Example.txt 已经提交到git仓库了。 现在，你想把这个文件名称改为 ./src/example.txt 。

我们知道，如果你右键重命名，在不修改git配置的前提下， 是不会有任何用的。

怎么办？

很简单，只需要使用下面命令进行修改即可： 

    git mv ./src/Example.txt ./src/example.txt

运行后执行 git status 就可以看见： 

On branch master
Your branch is up to date with 'origin/master'.

Changes to be committed:
  (use "git reset HEAD ..." to unstage)

        renamed:    src/Example.txt -> scr/example.txt

修改成功了，后续正常提交即可！
