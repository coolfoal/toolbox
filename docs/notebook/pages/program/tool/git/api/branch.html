分支

查看

查看分支状态，如果使用-r（即：git branch -r）表示查看远程分支状态，-a选项查看所有分支。 

git branch

创建

创建一个名称叫branchname的分支，此分支初始化拥有的是你创建此分支时活跃分支。 

git branch branchname

基于指定分支创建 

git branch branchname oldbranchname

基于指定提交创建 

git branch branchname hashname

切换活跃分支

切换当前活跃分支为branchname。 

git checkout branchname

删除

删除本地指定分支branchname。 

git branch -d branchname

删除远程指定分支branchname。 

git push hostname --delete branchname

改名

修改本地分支oldbranchname的名称为newbranchname。 

git branch -m oldbranchname newbranchname

合并

合并分支branchname到当前活跃分支。 

git merge branchname

变基

把分支branchname合并到当前活跃分支并以此为基。

温馨提示：和merge相比，不会留下合并记录。 

git rebase branchname
