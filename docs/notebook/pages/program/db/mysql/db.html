数据库操作

使用下面的命令进行登录： 

    mysql -h 主机名 -u 用户名 -p

回车后输入密码即可。

基础命令

查看版本 

    select version();

数据库操作

创建 

    create database [数据库名];

删除 

    drop database [数据库名];

选择

在你连接到 MySQL 数据库后，可能有多个可以操作的数据库，所以你需要选择你要操作的数据库： 

    use [数据库名];

查看

也就是查看当前有哪些数据库： 

    show databases;

当前

查看当前选中操作的数据库是谁： 

    select database();
