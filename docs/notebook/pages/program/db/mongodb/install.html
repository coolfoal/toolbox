安装和初始化

安装&启动

MacOS 

brew tap mongodb/brew
brew install mongodb-community@4.4

要将MongoDB（即mongod进程）作为macOS服务运行，执行以下命令： 

    brew services start mongodb-community@4.4

如果要停止mongod作为macOS服务运行： 

    brew services stop mongodb-community@4.4

此外，你还可以执行下面命令启动： 

    mongod --config /usr/local/etc/mongod.conf --fork
--fork表示在后台运行

配置path

执行过程中，mongod可能不存在，需要你配置PATH路径。如何确定bin文件夹？可以使用brew启动后，执行： 

    ps -ef|grep mongo

根据打印： 

501  8120     1   0 11:17上午 ??         0:01.86 /usr/local/opt/mongodb-community@4.4/bin/mongod --config /usr/local/etc/mongod.conf
501  8149  4020   0 11:19上午 ttys003    0:00.00 grep mongo

就可以确定PATH需要补充 /usr/local/opt/mongodb-community@4.4/bin ，也就是： 

export PATH=$PATH:/usr/local/opt/mongodb-community@4.4/bin

查看

查看 mongod 服务是否启动： 

    ps aux | grep -v grep | grep mongod

使用以上命令如果看到有 mongod 的记录表示运行成功。

停止程序

在 mongo 的命令出口中： 

> use admin
switched to db admin
> db.shutdownServer()
